## 0.0 Create-React-App(CRA) Configuration
- NodeJS 최신화
  - `nvm install --lts`
- CRA + TypeScript + Chakra.UI
    - `npx create-react-app . --template @chakra-ui/typescript`
- React-Router-Dom
    - `npm i react-router-dom`
- TanStack-Query
    - `npm i @tanstack/react-query axios @tanstack/react-query-devtools`
